# Todo Alarm

맥북용 할일 & 알림 데스크탑 앱

## 다운로드

[최신 버전 다운로드](https://github.com/bedcoding/todo-alarm/releases/latest)

> Apple Silicon(M1/M2/M3/M4) Mac 전용

## 설치 방법

1. `Todo-Alarm-x.x.x-mac-arm64.zip` 다운로드
2. 압축 해제
3. `Todo Alarm.app`을 Applications 폴더로 이동
4. 처음 실행 시 우클릭 → 열기 (보안 경고 우회)

## 주요 기능

- 일정 등록 및 알림
- 메모
- 이석 타이머
- 슬랙 연동
